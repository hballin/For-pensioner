# For pensioner
Brute-Force simulation for pensioner and other interests

It's often underestimated how quickly passwords taken from a dictionary can be cracked. And this is still a very common practice.

Through my volunteer work with retirees, this issue is ever-present.

For courses and workshops on general PC security, this tool is designed to demonstrate how easily simple passwords can be cracked. It decrypts and recovers both plaintext and encrypted passwords in no time at all.
